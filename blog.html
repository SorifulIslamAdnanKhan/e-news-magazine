var

The var is the oldest keyword for declaring a variable in JavaScript. It can be declared globally. That means you can access a var (variable ) everywhere in the program. You can declare the 2 variables with the same name using the var keyword. Also, the user can reassign the value into the var variable.


However, the scope of var is Global and Function. The scope is global when a var variable is declared outside a function. This means that any variable that is declared with var outside a function block is available for use in the whole window. var is function scoped when it is declared within a function. This means that it is available and can be accessed only within that function.
